== Description ==

Affix a Setmore Book now button easily in your WordPress site. The Setmore button would redirect your siteâ€™s visitors to the booking page of your Setmore account. Affix the button anywhere in the site premises without having to deal with/alter the code.

**Button Options**

* Button Title
* Open in a new window
* Overlay popup in the same window
* Wrap up the button to the side of the window
* Size
* Font Icon Type
* Icon Position
* Shape
* Color
* Background color
* Border color
* Title color
* Icon URL
* Custom CSS Class

**Insert Buttons Anywhere in your site**

Basically buttons can be inserted anywhere in your site

* Page/post
* In sidebars by using the SetMore Shortcode Widget
* In template files by using the button's shortcode and WP do_shortcode function
* Can create button for the staff

**Custom styling**

You can further apply styling to modify how the button will look on your site. To achieve it, you can

* View the shortcode of a button which has an exclusive CSS class to style other types of buttons.
* Create a new class for an instance of the button shortcode in the interface and style it according to your preference.

**Localization/Internationalization**

This plugin is localization/internalization based. It is compatible with different languages and can be adapted accordingly.
